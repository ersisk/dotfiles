# i3wm Config

My i3wm config file.

## Installation

copy config file to ~/.config/i3/

```bash
cp config ~/.config/i3/
```

## What you will need
[tilix](https://gnunn1.github.io/tilix-web/)
[Oh My Zsh](https://ohmyz.sh/)
[caffeine](https://www.atechtown.com/caffeine-for-ubuntu/)
